# **App Name**: AuraFit

## Core Features:

- Hero Section Display: Showcase a compelling hero section with a key message and primary call to action.
- About Me Section: Present a 'Sobre mí' section featuring a minimalist framed image and a personal bio.
- Services & Packages Section: Display various exercise packages within visually appealing, floating cards. Clicking a package redirects to a registration form.
- Package Registration Form: Provide a registration form for selected packages, including fields for Name, Email, Food Allergies, and Injury History.
- Result Gallery Section: Showcase 'Before and After' results in a gallery with rounded-3xl framed images or cards, maintaining the soft aesthetic.
- Soft Hover Effects: Implement subtle and elegant hover animations for interactive elements, enhancing user experience without distraction.
- Responsive Design: Ensure the landing page layout and elements adapt seamlessly across various screen sizes and devices.

## Style Guidelines:

- Primary background color: Soft Nude Pink (#F0D9DE). A gentle, muted pink for section backgrounds, setting a serene 'Soft Aesthetic' tone.
- Call to Action (CTA) color: Rose Pink (#E58FAB). This more vibrant pink is reserved for buttons and prominent interactive elements to ensure visibility and user engagement.
- Neutral background colors: Pure White (#FFFFFF) for primary canvas clarity and Cream (#FFFDD0) for secondary backgrounds or card fills, adhering to the user's request.
- Accent and decorative colors: Pastel Pink (#FADADD) for lighter text or subtle decorative touches. Gentle Lilac (#E1B9E1) as an analogous hue for subtle visual interest.
- Headline font: 'Playfair Display' (serif) for titles, conveying elegance and a high-end feel.
- Body font: 'PT Sans' (sans-serif) for body text, ensuring readability and a modern, friendly touch.
- Decorative font: 'Dancing Script' (handwriting/script style from Google Fonts) for ornamental text, quotes, or artistic details, adding a personal and elegant touch.
- Use delicate, outline-style icons with rounded forms to maintain the soft aesthetic, ensuring visual lightness and approachability.
- Clean and spacious layout with ample white space. Exercise package cards and 'Before & After' gallery items feature very rounded borders (rounded-3xl) and subtle floating effects. The 'About Me' image is minimally framed for focus.
- Subtle, smooth hover effects and transitions on interactive elements like buttons, cards, and gallery items, enhancing user engagement without being overly dramatic.
